# 公開資料與隱私提醒

這個 repo 最後可能會部署成公開網頁，因此請把 `profiles/*.json` 視為公開資料。

---

## 可以公開的內容

通常可以放：

- GitHub 帳號
- 暱稱或顯示名稱
- 一句話介紹
- 1 到 3 個興趣主題
- 想代表自己的 OS / 瀏覽器 / 編輯器陣營

---

## 不應該公開的內容

請不要放：

- 電話
- Email
- LINE
- Discord
- 學校、班級
- 住址
- 生日、年齡
- 不想公開的社群帳號
- 健康、宗教、政治、家庭背景等敏感資訊
- 任何你不希望陌生人看到的資訊

---

## 如果 Coding Agent 產生了不該公開的內容

請不要 commit。

你可以請 Agent：

```text
請移除 profiles/<我的 GitHub 帳號>.json 裡不適合公開的個人資料，只保留 github、displayName、tagline、interests 和 faction。
```

---

## GitHub 頭貼提醒

展示頁會使用 GitHub 頭貼：

```text
https://github.com/<github>.png
```

如果你的 GitHub 頭貼不想公開展示，請先更換 GitHub 頭貼，或詢問助教是否可以用替代帳號或測試帳號。
