# notes 資料夾

這裡放學員寫給 Coding Agent 看的自然語言草稿。

你可以從 `profile-note.template.md` 複製一份，改成自己的檔案：

```text
notes/<你的 GitHub 帳號>.md
```

例如：

```text
notes/octocat.md
```

---

## 為什麼不直接手寫 JSON？

因為這堂課要練習的是：

```text
我用自然語言描述偏好
→ Coding Agent 理解我的意思
→ Coding Agent 產生符合規格的 JSON
→ 我檢查 diff
```

JSON 是最後的產物，不是第一步的作業。

---

## 請填平常最常用或最想代表自己的陣營

因為今天使用的是電腦教室的電腦，所以請不要填「現在這台電腦」的環境。

請填：

- 你平常最常用的作業系統
- 你平常最常用的瀏覽器
- 你平常最常用或最想代表自己的編輯器 / IDE

這樣最後統計才會比較有趣。
